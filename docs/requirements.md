# LAN Drop 需求文档 (PRD)

## 1. 产品概述
LAN Drop 是一款专注于局域网内多设备协同的“零安装”传输工具。其核心目标是解决手机、平板、PC 之间碎片化文本（剪切板）和文件的高速流转问题。以类似微信“文件传输助手”的纯净聊天流交互形态，提供最符合直觉的使用体验。

## 2. 核心痛点与价值
*   **零安装门槛**：只需在 PC 端运行服务端，局域网内任何设备（iOS/Android/HarmonyOS/Mac）通过浏览器访问即可使用，免去全平台安装客户端的繁琐。
*   **多端身份识别**：打破传统局域网工具“分不清谁是谁”的困境，自动识别并标识不同设备的输入来源。
*   **全能传输聚合**：摒弃传统“发送/接收”的割裂式界面，将文本、图片、文件统一收拢至时间线（Timeline），符合现代人的交互直觉。

## 3. 功能需求详细说明

### 3.1 统一聊天流 (Unified Chat UI)
*   **全能输入**：在同一个输入框内支持发送文本、或点击“附件”按钮/拖拽上传文件。
*   **时间线混排**：将文本记录和文件记录按时间戳（timestamp/mtime）严格正序排列。
*   **状态隔离**：
    *   当前设备发出的消息/文件：显示在**右侧**，标记为“我”。
    *   其他设备发出的消息/文件/PC本地系统文件：显示在**左侧**，并附带明确的设备标识（如：💻 网页端 (Windows)、📱 平板 (iPad)、🖥️ 主机文件）。
*   **性能控制**：聊天窗口至多渲染最近 100 条记录，保证移动端浏览器的流畅度。

### 3.2 文本与剪切板共享
*   **快捷发送**：支持回车快捷发送文本。
*   **一键复制**：对于非自己发送的文本气泡，提供醒目的“📋 复制”按钮，并在复制成功后提供视觉反馈。

### 3.3 极速文件互传
*   **多文件与大文件上传**：支持一次性选择/拖拽多个文件，无格式和体积限制（基于局域网带宽上限）。
*   **乐观渲染与进度反馈**：文件开始上传时，聊天流中立即生成带有“⏳”和进度条的文件气泡，上传完成后平滑过渡为真实文件气泡。
*   **便捷下载**：点击历史流中的文件气泡即可直接触发浏览器高速下载。

## 4. 运行环境支持
*   服务端：Windows / macOS / Linux (需 Node.js 18+ 环境)
*   客户端：任意支持现代 HTML5 和 Fetch API 的主流浏览器 (Safari, Chrome, Edge, 微信内置浏览器等)
