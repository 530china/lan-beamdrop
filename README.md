# 🚀 LAN BeamDrop

**极速、极简的跨平台局域网文件与剪切板同步工具**

LAN BeamDrop 致力于打破操作系统之间的壁垒，在同一个局域网（Wi-Fi 或内网）下，无需安装任何客户端，即可实现 Windows、Mac、iOS、Android 和 HarmonyOS 之间的零延迟投送。

## ✨ 核心特性

- ⚡️ **开箱即用**：只需启动服务，局域网内任意设备通过浏览器（或扫码）即可无缝接入。
- 🖱️ **全局无缝拖拽**：支持文件拖拽上传与全局发光遮罩，一拖即传，并带有防误触机制。
- 📋 **剪切板与文本流**：在一台设备上复制，另一台设备瞬间可见。支持基于本地 JSON 的轻量级**持久化存储**，断电重启记录不丢失。
- 📁 **大文件分片流**：搭载智能前端分片与并发重试引擎，不再受限于内存大小，GB 级大文件稳定传输，直接跑满本地路由器极限带宽。
- 🔍 **极速极简检索**：纯前端内存级的性能优化，支持按“图片/视频”、“文档”、“链接”秒速分类检索，告别滑动眩晕。
- 💬 **设备群聊体验**：基于 WebSocket 增量实时推送，设备连接瞬间打通，像用微信群聊一样优雅地双向通讯。
- 🔒 **绝对私密**：数据 100% 在本地局域网内流动，无外网服务器中转，安全可靠。
- 🗂️ **跨平台资源管理器**：内置纯 Web 版目录选择器，支持快捷地址栏与新建文件夹，兼容 Win/Mac/Linux。
- ⚙️ **主机专属保护**：强大的本机专属高级设置面板，搭载严格的内网 IP 鉴权，支持一键清空历史与容量配置。
- 🚀 **测速与智能体检舱**：内置一键网络测速与综合诊断体检，3 秒全自动定位 VPN 绕路、2.4G 信号阻碍及服务端磁盘写入瓶颈。

## 🛠️ 快速开始

### 方案 A：小白免安装版（强烈推荐 ✨）

无需配置 Node 环境，下载后双击即可运行。

1. 前往 GitHub 的 [Releases 页面](../../releases/latest)。
2. 根据你的操作系统，下载对应的单文件可执行程序：
   - Windows 用户下载：`lan-beamdrop-win.exe`
   - macOS 用户下载：`lan-beamdrop-macos`
   - Linux 用户下载：`lan-beamdrop-linux`
3. 放在任意目录，直接双击运行（或在终端中运行）。
4. 控制台会打印出局域网访问地址及巨大的二维码。手机或平板直接扫码即可进入投送室！

### 方案 B：源码构建运行（开发者）

**环境要求**：[Node.js](https://nodejs.org/) (推荐 v18 及以上)

```bash
# 克隆项目
git clone https://github.com/530china/lan-beamdrop.git
cd lan-beamdrop

# 安装依赖
npm install

# 启动服务
npm start
```

## 📦 技术栈
- **后端**: Node.js + Express + WebSocket (ws)
- **前端**: 原生 ES6 Modules / HTML / CSS (毛玻璃拟态风)
- **发现**: mDNS 协议支持
- **测试**: Jest + Supertest (TDD 驱动开发)

## 🚀 自动构建与发版 (CI/CD)
本项目已配置高度自动化的 GitHub Actions 流水线。支持全平台（Windows、macOS、Linux）可执行文件 `pkg` 自动打包发布。

支持以下 **两种触发发版** 的姿势（流水线会自动拉取并打包上传附件）：
1. **产品经理流 (Releases 面板)**: 在 GitHub 的 `Releases` 页面点击 "Draft a new release"，填入带有 `v` 的版本号 Tag（如 `v1.0.0`），填写更新日志后，点击 "Publish release" 即可。打包完成后二进制附件会自动挂载到发布页。
2. **老板一键流 (Actions 面板)**: 进入 GitHub `Actions` 页面，选择 `CI/CD Pipeline`，点击 `Run workflow`，输入期望的版本号（如 `1.0.0`，不带 v）点击运行。脚本将全自动创建 Tag、打包并创建对应的 Release 页面。
